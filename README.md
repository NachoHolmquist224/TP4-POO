# Trabajo Práctico: Programación orientada a objetos

Este repositorio contiene la resolución de una serie de ejercicios prácticos orientados al manejo de POO, funciones y arrays en **JavaScript**.

## 🚀 Objetivo del Proyecto

El objetivo principal es demostrar el dominio en la manipulación de colecciones de datos y la creación de bloques de código reutilizables para optimizar la lógica de programación.

## 🛠️ Conceptos Aplicados

En este trabajo se han implementado los siguientes pilares de JS:

* **Funciones:** Declaración, funciones de flecha (arrow functions) y uso de parámetros/retornos.
* **Arrays (Arreglos):** Creación, acceso y modificación de elementos en listas.
* **Manipulación de Arrays:** Uso de métodos esenciales como `.push()`, `.pop()`, `.shift()`, `.unshift()` y `.find()`.
* **Uso de objetos:** Creacion, acceso y modificación de propiedades y metodos de un objeto.
